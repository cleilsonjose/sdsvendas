# 💻 Projeto
<strong>SdsVendas</strong> é um dashboard de vendas feito  na semana spring react - SDS 3.0

## 🛠 Tecnologias



## Backend
*  Java
* Spring Boot
* Spring Data JPA
* Spring Security
* Maven

## Frontend
* ReactJS
* TypeScript
* HTML
* CSS
* Bootstrap
* Apex Charts

## Implantação
* Backend: Heroku
* Front-end: Netlify
* Database: Postgresql 12

## 🚀 Como executar

### Back-End
Prerequisites: Java 11

```
# Clone repository
$ git clone https://github.com/GuilhermeProenca/projeto-sds3.git

# Enter the project back-end folder
$ cd backend

# Run project
$ ./mvnw spring-boot:run
```

## Front-End
Prerequisites: npm or yarn
```
# Clone repository
$ git clone https://github.com/GuilhermeProenca/projeto-sds3.git

# Enter the project front-end folder
$ cd frontend

# Install dependencies
$ yarn install

# Run project
$ yarn start

# Access localhost
http://localhost:3000
```
